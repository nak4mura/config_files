# 機能一覧作成コマンド

現在のプロジェクトから機能一覧を自動生成し、Markdown形式で出力します。

## 解析対象

- **APIエンドポイント**: REST API、ルーティング定義
- **ソースコード**: ビジネスロジック、サービス層、コントローラ層

## 出力内容

機能単位でカテゴライズされた一覧を生成します：

- API エンドポイントの一覧（HTTPメソッド、パス、説明、ファイル位置）
- 機能一覧（ドメイン/カテゴリごとのビジネスロジック）
- 統計情報（エンドポイント数、機能数など）

## 出力先

機能一覧は以下のパスに出力されます:
- `docs/features.md`

## 使用例

```bash
/list-features
```

## 実行フロー

list_features スキルを使用して以下の手順で実行：

1. プロジェクト構造の把握
2. API エンドポイントの抽出
3. ソースコードからの機能抽出
4. 機能のカテゴライズ
5. Markdown 形式での出力

## 注意点

- プロジェクトルートで実行してください
- docs/ ディレクトリが存在するか、作成できる必要があります
- 解析には時間がかかる場合があります
